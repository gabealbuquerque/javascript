Os tipos de dados em JavaScript podem ser:

  • Numéricos: Number
    • var temperatura = -1.5;
  • Cadeia de caracteres: String
    • var quarta = "Feijoada";
  • Booleanos: Boolean
    • var coxinha = true;
    • var caviar = false;
  • Arrays ou matrizes: sequência ou lista
    • var feijoada = ["arroz", "feijão preto", "carne seca", "paio", "toucinho", "couve", "farofa"];
